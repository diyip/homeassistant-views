Customize via URL Parameters

Both views can be customized by adding parameters after a ?. Use & to join multiple parameters.

https://.../index.html?name=My+Title&option=value

Spaces Use + for spaces, such as Car+Park+B1.
No quotation marks Write plain values such as true, false, or MEA.
Case-sensitive Parameter names should match exactly as documented.
Join multiple options Example: name=Site+A&show_date=true

Tip: Start with the default URL first. Once the view works, add one parameter at a time. This makes it easier to spot formatting mistakes.

Power Flow View

Shows live power flow in watts or kilowatts between major energy sources and loads.

What each item means

  • Grid — power drawn from or exported to the electricity grid.
  • Solar — current power generated by solar panels.
  • Home — total building or site consumption.
  • EV Chargers — charging load from electric vehicle chargers.
  • Non Fossil — percentage of electricity supplied by the grid that comes from renewable or low-carbon sources, based on external data.

Recommended Presets

Wall Display
Use slower animation and one decimal place for easier viewing from a distance.
?max_flow_rate=2&kw_decimals=1
Hide Non Fossil
Remove the Non Fossil indicator for a simpler layout.
?show_non_fossil=false
Hide EV Chargers
Remove the EV Chargers node when it is not relevant to your site.
?show_individual_0=false
Site-Specific Naming
Set a title and rename labels to match your site naming convention.
?name=Car+Park+B1&grid_name=Mains&solar_name=Rooftop+PV&home_name=Building+A

Notes and Limitations

  • Best viewed in a modern browser such as Chrome, Edge, or Safari.
  • The views auto-refresh automatically — Power Flow every 15 seconds, Energy Usage Graph every 5 minutes.
  • If a URL does not work as expected, remove parameters and test the default URL first.
  • For public display screens, use a stable network connection and disable browser sleep or screen timeout if needed.
